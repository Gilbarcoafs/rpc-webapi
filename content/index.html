rpc+ web api

the rpc+ webapi supports the following functions:

gpio functions

  • gpio export
  • gpio unexport
  • gpio direction get
  • gpio direction set
  • gpio value get
  • gpio value set

examples

  • export gpio 3: gpio/export?gpio=3
  • unexport gpio 3: gpio/unexport?gpio=3
  • set gpio 3 direction out: gpio/direction/set?gpio=3&direction=out
  • set gpio 3 direction in: gpio/direction/set?gpio=3&direction=in
  • get gpio 3 value: gpio/value/get?gpio=3
  • set gpio 3 value high: gpio/value/set?gpio=3&value=1
  • set gpio 3 value low: gpio/value/set?gpio=3&value=0
