WebApi

The RasPiComm+ WebApi exposes functionality of the Raspberry Pi, the RasPiComm+ and the its modules over a Rest WebApi.

It is a simple, open-source webapi that can be used out of the box on your raspberry pi without any dependencies. It even includes its own web-server, based on the open source webserver mongoose.

Support for the following sub systems are provided:

GPIOs

The following gpio actions are supported: export, unexport, get direction, set direction, get value, set value.

For examples and a live demo if you're running it on your raspberry pi, click here.
